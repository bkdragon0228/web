margin, padding

마진은 요소의 바깥여백을 패딩은 요소의 안쪽 여백을 지정하는 속성

border

보더는 요소에 대한 선을 적용할 때 사용하는 속성

width, style, color 한번에 선언 할 수 있다.

마진, 패딩, 보더의 공간 설정방향은 위에서부터 시계방향 순이다.

outline

아웃라인은 요소 바깥 주위에 둘러지는 선이며 만약 요소에 보더가 적용되어있다면 그 바깥에 생성된다.

너비, 높이

width -- 박스(요소) 너비

height -- 박스(요소) 높이

min-width --박스(요소)가 가질 수 있는 최소 너비

min-height -- 박스(요소)가 가질 수 있는 최소 최소 높이

max-width -- 박스(요소)가 가질 수있는 최소 최대 너비

max-height -- 박스(요소)가 가질 수 있는 최소 최대 높이
